# GALAXY Game

A small 2D space arcade game built with Python and Kivy.

## How to run
1. Install Python 3.9+
2. Install Kivy: pip install kivy
3. Run: python main.py

## Project structure
- main.py — entry point
- menu.py — menu logic
- user_actions.py — input handling
- transforms.py — helper functions
- *.kv — Kivy layout files
- images/, audio/, fonts/ — assets
